Web development voor smart connection

Voorbeelden van digital twins

  • Ik heb een webpagina gemaakt waarop de dagelijkse corona besmettingen, de dagelijkse corona doden en de totale corona doden in Nederland zijn te zien.

Lessen

Ik heb tijdens de eerste 6 weken van de minor elke week lessen gekregen over web development. Dit begon heel basic bij .html en werdt later uitgebreid met .css en .js. Deze website heb ik helemaal zelf gemaakt met een bootstrap, ik heb twee andere web pagina's gemaakt met een API erin. Deze zijn rechts te vinden. De ene is een webpagina waarop realtime het weer is te zien in druten. Verder geeft de afbeelding achter de temperatuur de omstandigheden weer. Er boven is een link naar de corona counter die ik heb gemaakt. Op deze webpagina zult u het aantal dagelijkse besmettingen, het aantal dagelijkse doden en het totaal aantal doden in Nederland vinden. Ook deze webpagina wordt volledig automatisch geupdate met de meest recente gegevens. Het vakje met het weer heb ik gemaakt met behulp en informatie van openweathermap. De webpagina heb ik gemaakt met de informatie van covid19api


Programma's

Ik heb voor het maken van deze website gebruik gemaakt van een bootstrap. Dit betekend eigenlijk dat je een al bestaande opzet aanpast naar eigen wensen. De bootstrap komt van TemplateMag. Een andere manier van website maken is via programma's als wordpress hierbij kan je als het waren je website tekenen en hoef je niet de code aan te passen. Ik heb dit echter niet gedaan, ik heb de "standaard code" die je krijgt bij de bootstrap aangepast. Het nadeel van een bootstrap is dat het vaak vrij standaard uitziende websites zijn. Het voordeel is dat je het standaard werk niet opnieuw hoeft te doen. Je kan namelijk ook een website vanuit het niets opbouwen maar de basis van een website is vrijwel altijd hetzelfde. Door een bootstrap te gebruiken bespaar je jezelf dus erg veel werk.
